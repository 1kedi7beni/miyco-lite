# Contributing to Miyco Lite

First off, thank you for considering contributing to Miyco Lite! It's people like you that make open source such a great community to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
Bugs are tracked as GitHub issues. When creating an issue, please explain the problem and include additional details to help maintainers reproduce the problem.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code lints.
5. Issue that pull request!

### Development Setup
1. Clone the repo: `git clone https://github.com/your-username/miyco-lite.git`
2. Install dependencies: `npm install`
3. Setup the database: `npx prisma db push && node prisma/seed.js`
4. Run the development server: `npm run dev`

Any contributions you make will be under the MIT Software License
